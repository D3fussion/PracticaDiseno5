# Mis Notas

> **Aplicación web para escribir y organizar tareas personales**
>
> [Ver la app en vivo](https://practica-diseno5.vercel.app)

---

## ¿Qué hace esta aplicación?

Es una libreta digital donde puedes **crear, editar y eliminar tareas** de forma sencilla. Está pensada para que se sienta como escribir en un cuaderno real: tiene líneas, márgenes, una barra lateral con el listado de tus tareas y navegación rápida para moverte entre ellas.

## Tecnologías usadas

| Herramienta            | Uso                                   |
| ---------------------- | ------------------------------------- |
| HTML + JavaScript      | Estructura y lógica de la interfaz    |
| Tailwind CSS + DaisyUI | Estilos y componentes visuales        |
| Vite                   | Servidor de desarrollo y empaquetado  |
| MongoDB + Mongoose     | Base de datos para guardar las tareas |
| Vercel                 | Despliegue en producción              |
